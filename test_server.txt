Após a instalação do servidor Ngnix, deve-se modificar seu arquivo de configuração da seguinte forma:

No diretório "/usr/local/nginx/conf", utilize o editor de texto para modificar o arquivo "nginx.conf".Use o comando "vi nginx.conf". Após a abertura do arquivo, copie o formato de configuração abaixo:

/-------------------------------------------------------------------------------------------------------------------------------------------/

worker_processes  auto;

error_log logs/error.log debug;

events {
    worker_connections  1024;
}

http {
    include       mime.types;
    default_type  application/octet-stream;

    sendfile        on;
    keepalive_timeout  65;

    server {
        listen       80;
        server_name  10.0.0.1;

        location / {
            root   /home/ubuntu/www;
        }

        error_page   500 502 503 504  /50x.html;
        location = /50x.html {
            root   html;
        }

    }
}

/-----------------------------------------------------------------------------------------------------------------------------------------------/

Agora copie o conteúdo do arquivo "server.txt" do github para o arquivo "/etc/nginx/sites-enabled/10.0.0.1" que você irá criar com o comando "vi 10.0.0.1" na localização exposta anteriormente.

Assim, será necessário iniciar o servidor a partir do comando: "service nginx start". Faça a verificação do estado do servidor com o seguinte comando: "service nginx status"
