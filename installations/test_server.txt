Após a instalação do servidor Ngnix, deve-se modificar seu arquivo de configuração da seguinte forma:

No diretório "/usr/local/nginx/conf", utilize o editor de texto para modificar o arquivo "nginx.conf".
Use o comando "vi nginx.conf". Após a abertura do arquivo, copie o formato de configuração abaixo:

/------------------------------------------------------------------------------------------------------------------------------/

worker_processes  auto;

error_log logs/error.log debug;

events {
    worker_connections  1024;
}

http {
    include       mime.types;
    default_type  application/octet-stream;

    sendfile        on;
    keepalive_timeout  65;

    server {
        listen       80;
        server_name  10.0.1.1;

        location / {
            root   /home/ubuntu/www;
        }

        error_page   500 502 503 504  /50x.html;
        location = /50x.html {
            root   html;
        }

    }
}

/------------------------------------------------------------------------------------------------------------------------------/


Agora desloque-se para o diretorio "/etc/nginx/sites-enabled" e crie um arquivo com o nome "10.0.0.1" com o comando 
"vi 10.0.1.1" e copie o seguinte código:

/------------------------------------------------------------------------------------------------------------------------------/

server {
        listen       80;
        server_name  10.0.1.1;

        location / {
            root   /home/ubuntu/www;
        }

        error_page   500 502 503 504  /50x.html;
        location = /50x.html {
            root   html;
        }
}

/------------------------------------------------------------------------------------------------------------------------------/

Após essa configuração inicial, será necessário iniciar o servidor a partir do comando: "service nginx start". 
Faça a verificação do estado do servidor com o seguinte comando: "service nginx status"
